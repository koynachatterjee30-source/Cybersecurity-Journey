# TCP/IP Model

## What is TCP/IP?
TCP/IP (Transmission Control Protocol/Internet Protocol) is a communication model used for connecting devices and transmitting data over networks and the internet.

## Layers of TCP/IP Model

1. Application Layer
   - HTTP
   - HTTPS
   - FTP
   - DNS
   - SMTP

2. Transport Layer
   - TCP
   - UDP

3. Internet Layer
   - IP Addressing
   - Routing

4. Network Access Layer
   - Ethernet
   - MAC Address

TCP:
- Connection-oriented
- Reliable
- Error checking available
